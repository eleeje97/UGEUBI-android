<h3>소개</h3>
집에서 방치되는 약을 효율적으로 관리 및 복용할 수 있게 도와주는 서비스
<br/>
<hr/>

<h3> 주요 서비스</h3>
1. 복용시간 및 요일 정보를 알림 서비스로 제공 <br/>
2. 잊기 쉬운 약의 유통기한을 관리하여 복용할 수 있는 서비스 제공 <br/>
3. 다양한 약에 대한 정보를 한 곳에서 확인 할 수 있는 서비스 제공 <br/>
